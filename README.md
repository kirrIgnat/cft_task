Работа выполнена в рамках тестового задания для компании Центр Финансовых Технологий


сборка осуществяется через makefile

компилятор -- ```gcc -std=c11 -Wall -Wextra -O2```
```
make
```

на выходе 2 исполняемых файла 
task -- основная утилита
task_test -- тестирование 

запуск
```
./task <in_fliename1> <in_fliename2> <out_filename>
./task_test
```

модули и назначение 
1) mian.c -- точка входа
2) io.c -- save load
3) core.c -- join sort
4) util.c -- сторонние функции, у нас это печать
